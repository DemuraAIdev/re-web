# How to make like this
![Count](https://count.vahryiskandar.my.id/get/@cyberai2323?theme=rule34)

just add this to your markdown html or svg
```js:index.html
<img src="https://count.getloli.com/get/@:name" alt=":name" />
```
```svg
<svg xmlns="http://www.w3.org/2000/svg" width="100" height="20">
  <foreignObject width="100%" height="100%">
    <img src="https://count.getloli.com/get/@:name" alt=":name" />
  </foreignObject>
</svg>
```
```md
![Count](https://count.getloli.com/get/@:name)
```
Just replace `:name` with your name

# theme
You can change theme by add `theme` query
```html
<img src="https://count.getloli.com/get/@:name?theme=rule34" alt=":name" />
```
```svg
<svg xmlns="http://www.w3.org/2000/svg" width="100" height="20">
  <foreignObject width="100%" height="100%">
    <img src="https://count.getloli.com/get/@:name?theme=rule34" alt=":name" />
  </foreignObject>
</svg>
```
```md
![Count](https://count.getloli.com/get/@:name?theme=rule34)
```
